Introducción a la Programación

La programación es el proceso de escribir instrucciones que permiten a las computadoras realizar tareas específicas. A través del código, se crean aplicaciones, se automatizan procesos y se resuelven problemas de manera eficiente. Es un campo bastante amplio, con diferentes enfoques y lenguajes, cada uno diseñado para abordar distintas necesidades y tipos de proyectos. Al aprender a programar, puedes desarrollar soluciones tecnológicas que transforman ideas en herramientas útiles, facilitando tareas y mejorando la productividad en muchos ámbitos.

Este espacio está diseñado para ofrecerte una visión básica de la programación, cubriendo los fundamentos esenciales de forma simple y concisa. A lo largo de los contenidos, verás una breve explicación de los temas más importantes y tendrás acceso a tutoriales prácticos que te ayudarán a poner en práctica lo aprendido. Si eres nuevo en el mundo de la programación o quieres refrescar conceptos básicos, aquí encontrarás el punto de partida ideal.

¿Qué es la Programación Estructurada?

La programación estructurada es un estilo de programación que organiza el flujo de ejecución del programa de manera lógica, permitiendo que sea más comprensible y fácil de mantener. A diferencia de otros paradigmas como la programación orientada a objetos, se enfoca en el uso de estructuras de control y la organización secuencial del código

Principales características:

Secuencia: El código se ejecuta en orden, línea por línea, sin saltos inesperados.
Selección: El flujo del programa puede tomar diferentes caminos dependiendo de las condiciones (estructura if).
Iteración: Permite repetir acciones utilizando bucles (for, while).

Variables y Tipos de datos

Las variables son contenedores que almacenan datos y que pueden cambiar durante la ejecución del programa. Para declarar una variable, es necesario especificar el tipo de dato (números, caracteres, booleanos, etc.) que almacenará y un nombre único para la variable.

Tipos de Datos:

Entero (int): Almacena números enteros.
Flotante (float): Almacena números con decimales.
Doble (double): Similar a float, pero con mayor precisión.
Carácter (char): Almacena un solo carácter.
Booleano (bool): Solo puede ser true o false.

Ejemplo: 

int edad = 21; // Variable de tipo entero
float altura = 1.55; // Variable de tipo flotante
char text = 'Hola mundo'; // Variable de tipo carácter
bool esEstudiante = true; // Variable de tipo booleano"

Estructuras de Control

Las estructuras de control son fundamentales en la programación estructurada. Permiten decidir qué instrucciones se deben ejecutar en función de ciertas condiciones o repetir bloques de código. Se dividen en estructuras condicionales y estructuras iterativas (bucles).

Condicionales (if, else, switch):
Las estructuras condicionales permiten ejecutar diferentes bloques de código según se cumpla o no una condición.

• if - else: Se usa para ejecutar un bloque de código si una condición es verdadera, y otro si es falsa.

int main() {
    int edad = 18;

    if (edad >= 18) {
        printf("Eres mayor de edad.\n");
    } else {
        printf("Eres menor de edad.\n");
    }

    return 0;
}

• if - else if - else: Se usa cuando hay múltiples condiciones a evaluar en cascada.

int main() {
    int nota = 75;

    if (nota >= 90) {
        printf("Tienes una A.\n");
    } else if (nota >= 80) {
        printf("Tienes una B.\n");
    } else if (nota >= 70) {
        printf("Tienes una C.\n");
    } else {
        printf("Reprobaste.\n");
    }

    return 0;
}

• switch - case: Se usa cuando hay varias opciones específicas para una variable.

int main() {
    int opcion = 2;

    switch (opcion) {
        case 1:
            printf("Seleccionaste opción 1.\n");
            break;
        case 2:
            printf("Seleccionaste opción 2.\n");
            break;
        case 3:
            printf("Seleccionaste opción 3.\n");
            break;
        default:
            printf("Opción no válida.\n");
    }

    return 0;
}

Bucles (for, while, do while):
Los bucles permiten ejecutar un bloque de código varias veces mientras se cumpla una condición.

• while: Ejecuta un bloque de código mientras la condición sea verdadera.

int main() {
    int i = 1;

    while (i <= 5) {
        printf("Iteración %d\n", i);
        i++;
    }

    return 0;
}

• do - while: Similar a while, pero siempre ejecuta el bloque al menos una vez.

int main() {
    int i = 1;

    do {
        printf("Iteración %d\n", i);
        i++;
    } while (i <= 5);

    return 0;
}

• for: Se usa cuando conocemos cuántas veces debe repetirse el bloque de código.

int main() {
    for (int i = 1; i <= 5; i++) {
        printf("Iteración %d\n", i);
    }

    return 0;
}
